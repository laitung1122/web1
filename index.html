ຄໍາຖາມ 1: ຮອກໂມນເເມ່ນຫຍັງ?
- ຮອກໂມນ ເເມ່ນທາດເຄມີປະເພດຕ່າງໆ ເຊັ່ນ:ໂປຮຕີນ(protien), ອາມີນ(Amin) ເເລະ ອາມີໂນອາຊິດ(Amino_Acid) ທີ່ຜະລິດຂຶ້ນຈາກຕ່ອມປ່ອຍໃນ ເເລະ ເເພຈຸລັງປະເພດຕ່າງໆ. (will update soon for more infomation)
ຄໍາຖາມ 2: ຕ່ອມປ່ອຍໃນມີຕ່ອມປະເພດໃດເເນ່?ເເຕ່ລະຕ່ອມເປັນຄືເເນວໃດ?
- ຕ່ອມປ່ອຍໃນມີ 9 ປະເພດຄື:
+ ຕ່ອມເທິງສະໝອງ
+ ຕ່ອມໃຕ້ສະໝອງ
+ ຕ່ອມຄໍຫອຍ
+ ຕ່ອມອ້ອມຄໍຫອຍ
+ ຕ່ອມໄທມັສ
+ ຕ່ອມເທິງໝາກໄຂ່ຫຼັງ
+ ຕ່ອມມ້າມ
+ ຕ່ອມຮວຍໄຂ່
+ ຕ່ອມອັນທະ.
ຄໍາຖາມ 3: ຕ່ອມເທິງສະໝອງມີຫນ້າທີ່ຫຍັງ?
- ຕ່ອມເທິງສະໝອງມີໜ້າທີ່ໃນການຜະລິດຮອກໂມນ"ເມລາໂຕນິນ(melatonin)"ເພື່ອມາຄວບຄຸມອະໄວຍະວະທາງເພດໃຫ້ຂະຫຍາຍຕົວຕາມປົກກະຕີໂດຍສະເພາະເເມ່ນໃນຊ່ວງທີ່ຈະກ້າວເປັນໄວໜຸ່ມສາວ.
ຄໍາຖາມ 4: ຕ່ອມໃຕ້ສະໝອງມີຫນ້າທີ່ຫຍັງ?
- ຕ່ອມໃຕ້ສະໝອງມີໜ້າທີ່ໃນການຜະລິດຮອກໂມນຫຼາຍຊະນິດ ເພື່ອຄວບຄຸມການເຮັດວຽກຂອງຕ່ອມປ່ອຍໃນປະເພດອື່ນໆ ເເລະ ອະໄວຍະວະຕ່າງໆຂອງຮ່າງກາຍ.
ຄໍາຖາມ 5: ຕ່ອມໃຕ້ສະໝອງເເບ່ງອອກເປັນຈັກເຂດຫຍ່ອຍ?ຄືເຂດໃດເເນ່?
- ຕ່ອມໃຕ້ສະໝອງເເບ່ງອອກເປັນ 3 ເຂດຍ່ອຍຄື:
+ ເຂດເບື້ອງໜ້າ
+ ເຂດເບື້ອງກາງ
+ ເຂດເບື້ອງຫຼັງ.
ຄໍາຖາມ 6: ຮອກໂມນຊະນິດໃດມີຫນ້າທີ່ກະຕຸ້ນຮ່ວຍໄຂ່ຂອງເພດຍິງ ເເລະ ອັນທະເພດຊາຍ,ການຜະລິດອະສຸຈິຂອງເພດຊາຍ?
- ຮອກໂມນທີ່ມີໜ້າທີ່ໃນການກະຕຸ້ນຮ່ວຍໄຂ່ຂອງເພດຍິງເເມ່ນ"ອິສໂຕເຈນ(estrogens)" ສ່ວນອັນທະຂອງເພດຊາຍ ເເລະ ການຜະລິດນຳ້ອະສຸຈິຂອງເພດຊາຍເເມ່ນ "ເເອນໂຕເຈນ(Androgens)".
ຄໍາຖາມ 7: ຮອກໂມນລູທີໄນຊິງ(Luteinizing)ມີໜ້າທີ່ຫຍັງ?
- ຮອກໂມນລູທີໄນຊິ້ງມີໜ້າທີ່ໄນການກະຕຸ້ນຕ່ອມອັນທະ ເເລະ ຮ່ວຍໄຂ່.
ຄໍາຖາມ 8: ຮອກໂມນຊະນິດໃດທີ່ມີໜ້າທີ່ກະຕຸ້ນຕ່ອມນ້ຳນົມເພດຍິງ?
- ຮອກໂມນທີ່ກະຕຸ້ນຕ່ອມນຳ້ນົມຂອງເພດຍິງເເມ່ນຮອກໂມນ"ໂພຣເເລກຕິນ(Prolactin)".
ຄໍາຖາມ 9: ຖ້າຕ່ອມຄໍຫອຍເຮັດວຽກຜິດປົກກະຕິຈະມີຜົນກະທົບເເນວໄດ?
- ຖ້າຕ່ອມຄໍຫອຍເຮັດວຽກຜິດປົກກະຕິມັນຈະສົ່ງຜົນຕໍ່ການຄວບຄຸມປະລິມານທາດ"ໄອໂອດິນ"ໃນຮ່າງກາຍໄດ້ ເເລະ ຖ້າຂາດທາດໄອໂອດິນຫຼາຍເກີນໄປຈະສົ່ງຜົນສະທ້ອນຕໍ່ການເຕີບໄຫຍ່ຂອງຮ່າງກາຍ, ສະຕິປັນຍາເຊັ່ນ: ເຕີບໄຫຍ່ຊ້າ, ຄວາມຈຳເຊື່ອມ, ເປັນຄໍໜຽງ ເເລະ ອື່ນໆ.
ຄໍາຖາມ 10: ຕ່ອມອອ້ມຄໍຫອຍມີຫນ້າທີ່ເປັນເເນວໃດ?
- ຕ່ອມອ້ອມຄໍຫອຍມີໜ້າທີ່ໃນການຜະລິດຮອກໂມນ"ປາຣາ(Para)"ເຊິ່ງມີໜ້າທີ່ໃນການຄວບຄຸມປະລິມານເເຄວຊ້ຽມໃນເສັ້ນເລືອດ ເເລະ ກະດູກໃຫ້ຢູ່ໃນປະລິມານຄົງທີ່.
ຄໍາຖາມ 11: ຕ່ອມໄທມັສ(Thymus)ມີໜ້າທີ່ເເນວໃດ?
- ຕ່ອມໄທມັສມີໜ້າທີ່ໃນການຜະລິດຮອກໂມນ"ທີລິມໂຟໄຊ"ທີ່ມີບົດບາດສໍາຄັນຕໍ່ການເຕີບໃຫຍ່ໃນໄວເດັກເເລະປ້ອງກັນຮ່າງກາຍ.
ຄໍາຖາມ 12: ຕ່ອມມ້າມເຮັດໜ້າທີ່ຫຍັງ?(ເວົ້າໃຫ້ລະອຽດ)
ຕ່ອມມ້າມມີໜ້າທີ່ທັງເປັນຕ່ອມປ່ອຍນອກເເລະຕ່ອມປ່ອຍໃນ.
+ ຕ່ອມປ່ອຍນອກມີໜ້າທີ່: ໃນການຜະລີດນຳ້ຍ່ອຍເພື່ອຊ່ວຍໃນການຍ່ອຍອາຫານບໍລີເວນລຳໄສ້ອອ່ນ.
+ ຕ່ອມປ່ອຍໃນມີໜ້າທີ່: ໃນການຜະລິດຮອກໂມນ"ອິນຊູລິນ"ເພື່ອຄວບຄຸມປະລິມານນ້ຳຕານໃນເລືອດໃຫ້ຢູ່ໃນພາວະປົກກະຕິ.
ຄໍາຖາມ 13: ຕ່ອມເທິງໝາກໄຂ່ຫຼັງມີຈັກຊັ້ນ ເເລະ ບອກໜ້າທີ່ຂອງເເຕ່ລະຊັ້ນ?
ຕ່ອມເທິງໝາກໄຂ່ຫຼັງມີ 2 ຊັ້ນຄື: ຊັ້ນນອກ ເເລະ ຊັ້ນໃນ.
+ ຊັ້ນນອກມີໜ້າທີ່: ໃນການຜະລິດຮອກໂມນ"ກຣູໂຄຣຄໍຕິຄອຍ(Clucocorticoid)" ເເລະ ຮອກໂມນ"ມິເນຣາໂລຄຣໍຕິຄອຍ(mineralocorticoid)"ເພື່ອຄວບຄຸມການເເລກປ່ຽນປະລີມານນຳ້ຕານ,ເກືອ ເເລະ ນ້ຳໃນຮ່າງກາຍໃຫ້ຄົງທີ່.
+ ຊັ້ນໃນມີໜ້າທີ່: ໃນການຜະລິດຮອກໂມນ"ອາດຣີນາລິນ(Adrenalin)"ເເລະຮອກໂມນ"ນໍຣາເດຣີນາລິນ(Noradrenalin)"ມີໜ້າທີ່ໃນການກະຕຸ້ນການບິບຕົວຂອງເສັ້ນເລືອດ, ຫົວໃຈ, ກ້າມ ເເລະ ການເຮັດວຽກຂອງຕັບ.
ຄໍາຖາມ 14: ຕ່ອມອັນທະກັບຕ່ອມຫ່ວຍໄຂ່ເຮັດໜ້າທີ່ຫຍັງ?
+ ຕ່ອມອັນທະເຮັດໜ້າທີ່ໃນການຜະລິດຈຸລັງອະສຸຈິຂອງເພດຊາຍ(ອະສຸຈິ).
+ ຕ່ອມຮ່ວຍໄຂ່ມີໜ້າທີ່ໃນການຜະລິດຈຸລັງສຶບເຊື້ອຂອງເພດຍິງ(ຮ່ວຍໄຂ່).
ຄໍາຖາມ 15: ສິ່ງທີ່ເປັນຜົນກະທົບຕໍ່ການເຮັດວຽກຂອງຮອກໂມນມີຫຍັງເເດ່?
- ສິ່ງທີ່ເປັນຜົນກະທົບຕໍ່ການເຮັດວຽກຂອງຮອກໂມນເເມ່ນການທີ່ຮ່າງການຂອງຄົນເຮົາພົວພັນກັນທາງດ້ານເຄມີ.
ຄໍາຖາມ 16: ຖ້າປະລິມານນໍ້າທີ່ຮັບເຂົ້າແລະຂັບອອກບໍ່ສົມດູນຈະມີບັນຫາກັຍງເກີດຂື້ນແກ່ຮ່າງກາຍ?
- ຖ້າປະລິມານນຳ້ທີ່ຮັບເຂົ້າ ເເລະ ຂັບອອກບໍ່ສົມດຸນຮ່າງກາຍກໍຈະເກີດມີອາການ: ຕົວເຫຼືອງ, ຂາດນ້ຳ, ເລືອດຂຸ້ນ ເເລະ ພາວະຂາດນຳ້ນັ້ນຈະສົ່ງຜົນກະທົບຕໍ່ການເກີດພະຍາດທາງໝາກໄຂ່ຫຼັງ. (Answer by Mr.Kham Sinh)
ຄໍາຖາມ 17: ຮ່າງກາຍຂອງຄົນເຮົາກຳຈັດທາດໃດແດ່ອອກຮ່າງກາຍ?
ທາດທີ່ຮ່າງກາຍຂອງຄົນເຮົາກຳຈັດອອກມີ:
+ ນ້ຳ
+ ຢູເຣຍ
+ ກົດຢູຣິກ
+ ເເອມໂມນຽນ
+ ອີອົງຕ່າງໆຄື: ອີອົງໂຊດຽມ ເເລະ ຄຣໍດາຍອີອົງ.
ຄໍາຖາມ 18: ມີທາດໃດແດ່ທີ່ທໍ່ໜ່ວຍໝາກໄຂ່ຫັລງດູດກັບ?
- ທາດທີ່ທໍ່ໜ່ວຍໝາກໄຂ່ຫຼັງດູດກັບມີ:
+ ນ້ຳ
+ ກູໂກສ
+ ໂປຣຕີນ
+ ອາຊິດອາມີໂນ
ຄໍາຖາມ 19: ຄວາມເຂັ້ມຂອງທາດຕ່າງຯໃນນໍ້າເລືອດຫຼືປັດສະວະມີຄວາມຄົງທີ່ສະເໝີບໍ່?
ໃນຂະນະທີ່ຮ່າງກາຍປົກກະຕິຮ່າງກາຍສາມາດຄວບຄຸມຄວາມເຂັ້ມຂອງທາດຕ່າງໆໃນນ້ຳເລືອດ ເເລະ ປັດສະວະໃຫ້ຄົງທີ່ໄດ້.
📌Tips: ເເຕ່ຖ້າຮ່າງກາຍບໍ່ປົກກະຕິຄວາມເຂັ້ມຂຸ້ນຂອງທາດຕ່າງໆໃນນ້ຳເລືອດເເລະປັກສະວະກໍຈະບໍ່ຄົງທີ່.
ຄໍາຖາມ 20: ການເຮັດວງກຂອງນໍ້າຍ່ອຍຕໍ່າຈະມີຜົນແນວໃດ?
- ການເຮັດວຽກຂອງນ້ຳຍ່ອຍຕ່ຳຈະມີຜົນຕໍ່ປະຕິກິລິຍາເຄມີທີ່ກ່ຽວຂ້ອງກັບນ້ຳຍ່ອຍຕົວນັ້ນຈະເກີດໄດ້ຊ້າເຊິ່ງຈະສົ່ງຜົນຕໍ່ອະໄວຍະວະ ຫຼື ຮ່າງກາຍສ່ວນທີ່ກ່ຽວຂ້ອງກັບທາດຈາກປະຕິກີລິຍານັ້ນ.
ຄໍາຖາມ 21: ຮ່່ງງກາຍມີຄວາມຈໍາເປັນຫຼືບໍ່ທີ່ຕ້ອງການຮັກສາຄວາມສົມດູນລະດັບກົດ-ດັງໃຫ້ເໝາະສົມ?
- ຮ່າງກາຍມີຄວາມຈຳເປັນຢ່າງຍິ່ງທີ່ຕ້ອງຮັກສາກົດ-ດັ່ງໃຫ້ເໝາະສົມນັ້ນເເມ່ນໃຫ້ການເຮັດວຽກຂອງລະບົບຕ່າງໆໃນຮ່າງກາຍດຳເນີນໄປດ້ວຍດີ
ຄໍາຖາມ 22: ຍ້ອນຍັງຈຶ່ງຕ້ອງຮັກສາຄວາມສົມດູນຄວາມເປັນກົດແລະດັງຂອງເລືອດ?
- ເນື່ອງຈາກຄວາມເຂັ້ມຂຸ້ນຂອງH+ທີ່ເໝາະສົມຈະຊ່ວຍຮັກສາໂປຣຕີນໄວ້ທີ່ສາມາດເຮັດວຽກໄດ້ຢ່າງມີປະສິດທີພາບສູງສຸດ.ໃນເລືອດມີໂປຣຕີນ ເເລະ ນຳ້ຍ່ອຍຫຼາຍຊະນິດເຊິ່ງສາມາດເຮັດວຽກໄດ້ໃນລະດັບpHປະມານ 7.
📌Tip: ດັ່ງນັ້ນຖ້າມີການປ່ຽນເເປງຄວາມເຂັ້ມຂຸ້ນຂອງ H+ໃນເລືອດຈະພາໃຫ້ປະສິດທິພາບໃນການເຮັດວຽກຂອງໂປຣຕີນ ເເລະ ນ້ຳຍ່ອຍຕ່າງໆໃນເລືອດຫຼຸດລົງ ເເລະ ສົ່ງຜົນເສຍຕໍ່ຮ່າງກາຍໃນທີ່ສຸດຈຶ່ງຈຳເປັນຕ້ອງຮັກສາຄວາມສົມດຸນຄວາມເປັນກົດ-ດັ່ງໃນເລືອດໃຫ້ຄົງທີ່.
ຄໍາຖາມ 23: ຄົນເຮົາມີການຮັກສາຄວາມຮ້ອນໃຫ້ຮ່າງກາຍແມ່ນສະໝອງສວ່ນໃດເຮັດໜ້າທີ່ຮັກສາ?
- ຄົນເຮົາສາມາດຮັກສາຄວາມຮ້ອນໄວ້ໃນຮ່າງກາຍໄດ້ໂດຍເເມ່ນສະໝອງສ່ວນ"ໄຮໂພທາລາມາສ"
ຄໍາຖາມ 24: ຍ້ອນສາເຫດອັນໃດເມື່ອເຮົາຮູ້ສຶກຮ້ອນຈິ່ງຮູ້ສຶກຫິວນໍ້າໄປນໍ້າ?
- ເມື່ອເຮົາຮູ້ສຶກຮ້ອນຈະຮູ້ສຶກຫິວນ້ຳໄປນຳນັ້ນເເມ່ນເນື່ອງຈາກມີການລະບາຍຄວາມຮ້ອນອອກຈາກຮ່າງກາຍໂດຍການຂັບເຫື່ອອອກມາເພື່ອຮັກສາອຸນຫະພູມໃນຮ່າງກາຍໃຫ້ຄົງທີ່ພາໃຫ້ຮ່າງກາຍສູນເສຍນ້ຳໄປກັບເຫື່ອເເລ້ວສົ່ງຜົນໄຫ້ຄວາມເຂັ້ມຂຸ້ນຂອງເລືອດສູງຂຶ້ນຈຶ່ງມີການສົ່ງສັນຍານໄປກະຕຸ້ນສູນຄວບຄຸມຄວາມຫິວນ້ຳທີ່ໄຮໂພທາລາມາສເຮົາຈຶ່ງຮູ້ສຶກຫິວນ້ຳ.
ຄໍາຖາມ 25: ມີທາດໃດແດ່ນອກຈາກນໍ້າແລ້ວທີ່ຮ່າງກາຍຕ້ອງການ?
- ນອກຈາກນ້ຳເເລ້ວຮ່າງກາຍຍັງຕ່ອງການທາດ: ເເຄວຊ້ຽມ, ໂປຣຕີນ, ວິຕາມິນ, ນາຕີ(Na), ກຣໍຣົວ(Cl).
ຄໍາຖາມ 26: ຜິວໜັງມີຄວາມສໍາຄັນຕໍ່ການຄວບຄຸມອຸນຫະພູມຂອງຮ່າງກາຍແນວໃດ?
- ບໍລິເວນຜິວໜັງມີອະໄວຍະວະທີ່ກ່ຽວຂ້ອງກັບການຄວບຄຸມອຸນຫະພູມຮ່າງກາຍມີ:ເສັ້ນຂົນ ເເລະ ກ້າມຊີ້ນຈ່ອງເສັ້ນຂົນ, ເສັ້ນເລືອດ,ຕ່ອມເຫື່ອ
📌Tip: ເມື່ອສະພາບເເວດລ້ອມປ່ຽນເເປງອະໄວຍະວະເຫຼົ່ານີ້ເຮັດວຽກປະສານກັນເພື່ອປັບອຸນຫະພູມພາຍໃນໃຫ້ຄົງທີ່ເຊັ່ນ:ເມື່ອອຸນຫະພູມຫຼຸດລົງກ້າມຊີ້ນຈະຈ່ອງເສັ້ນຂົນຖືກກະຕຸ້ນໃຫ້ຫົດຕົວດຶງເສັ້ນຂົນໃຫ້ຕັ້ງຂຶ້ນເພື່ອກັນອາກາດໄວ້
ຄໍາຖາມ 27: ຈົ່ງຍົກຕົວຢ່າງການລະບາຍຄວາມຮ້ອນແລະປ້ອງກັນການສູນເສຍຄວາມຮ້ອນອອກຮ່າງກາຍ?
- ສັດມີວິທີການລະບາຍຄວາມຮ້ອນອອກຈາກຮ່າງກາຍໄດ້ຫຼາຍຮູບເເບບເຊັ່ນ:ຄວາຍລົງໄປເເຊ່ນ້ຳ, ຊ້າງຈະໃຊ້ງວງສູບນ້ຳເເລະພົ່ນໃສ່ຮ່າງກາຍ, ໝາຈະເຫັນມັນເເລບລີ້ນຕະຫຼອດເວລາທີ່ມີອາກາດຮ້ອນເອົ້າ ເເລະ ອື່ນໆ.
